Procedure to Create and View a Website on the Production Server

  1. Create your html file. Do NOT forget to add the doctype html (preceded by an exclamation point!!) to the very beginning of your html.
  2. Link your css file to your html file in the head section using this: link rel="stylesheet" href="NAMEOFTHECSSFILE" type="text/css" (add < /> around it !)
  3. Fill in the rest of your html, making sure to close the tags.
  4. Style your css sheet. Yep. Always put a semicolon after each element!
  5. Presuming you've already opened a server (python -m SimpleHTTPServer) , go to the production server by typing the command "sftp"
  6. Make sure you are in your repository folder! Then, add each file with the command "put FILENAME".
